# Support

Use [GitHub Issues](https://github.com/nishu-builder/git-vdb/issues) for
reproducible bugs and focused feature proposals. Search existing issues first
and include the `git-vdb` version or revision, operating system, Git version,
repository kind, command or API call, and a minimal reproduction.

This is a community-maintained open-source project. Support is best effort and
has no guaranteed response time. General Rust, Git, embedding-model, and
application architecture questions may be better suited to their respective
communities.

Do not post secrets, private payloads, proprietary vectors, or full production
repositories. For vulnerabilities, follow [SECURITY.md](SECURITY.md) and report
privately.
